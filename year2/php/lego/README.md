## Lego World Doetinchem

### How to run

The entire application can be run using
[Sail](https://laravel.com/docs/10.x/sail). Please make sure you have
[Docker](https://www.docker.com/) installed.

```sh
./vendor/bin/sail up
```

## Technical specs

This is a laravel application running inside a docker container using sail.
It uses postgresql for its database, and vite for asset handeling inside bun.
And it runs using laravel octaine to run nice and fast.
